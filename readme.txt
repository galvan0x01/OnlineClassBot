About:
	This script can attend all of your google meet classes and can run forever

Steps to run this script on your machine:
	Initialize your mysql database
	Create a database and create a table called "timetable"
	Enter your timetable in the table name "timetable"
	Name all your free periods as "free" in the table
	Enter the host, username, password, database in the python script
	Enter the class codes of the respective classes in the python script
